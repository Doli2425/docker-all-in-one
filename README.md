# docker-all-in-one

Docker 기초부터 이미지 빌드, 멀티스테이지, Compose, HAProxy, Minecraft 서버까지  
여러 실습을 진행하며 정리한 개인 학습용 저장소입니다.

## 📂 구성
- `05_2048` : 2048 게임 서비스 Docker 실습
- `05_dockerfile` : Dockerfile 기본 문법 실습
- `05_ImageLayer` : 이미지 레이어 구조 분석
- `05_MultiStage` : 멀티 스테이지 빌드 실습
- `05_onbuild` : ONBUILD 이미지 테스트
- `05_python` : Python 앱 컨테이너 이미지 실습
- `05_shellscript` : 셸 스크립트 기반 ENTRYPOINT 테스트
- `05_t_rex_runner` : T-Rex 게임 서비스 실습
- `07_compose` : Docker Compose 기본 실습
- `07_compose-cmd` : command/entrypoint 구조 확인
- `07_haproxy-compose` : HAProxy + Backend Compose 구성
- `07_haproxy-cmd` : HAProxy 단독 설정 테스트
- `07_minecraft` : Minecraft 서버 Docker 실습
- `mail` : mail server 관련 테스트

## 🎯 목적
- Docker 전체 기능 흐름을 실습으로 익히기
- 이미지 빌드/레이어 이해
- Compose 기반 멀티 컨테이너 구성 연습
- 실습 코드 보관 및 복습용
